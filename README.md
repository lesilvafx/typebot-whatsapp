# Tema WhatsApp para Typebot

Este projeto contém um script que transforma o Typebot em um layout parecido com o WhatsApp, detectando automaticamente:

- iOS (tema claro e escuro)
- Android (tema claro e escuro)
- Desktop
- Tema claro/escuro do sistema

O script adiciona:
- Barra superior personalizada estilo WhatsApp
- Avatar, nome, ícone de verificação e status (online/digitando/gravando)
- Mensagem inicial personalizada
- Sons de notificação
- Marcação de horário nas mensagens
- Player de áudio customizado com WaveSurfer
- Estilos especiais para cada sistema operacional e tema
